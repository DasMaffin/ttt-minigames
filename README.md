# TTT Minigames
TTT Minigames (Browser Games) in a small Popup

## ConVars
notify_minigames 0 --Value 0 or 1. 0 = No message on death for typing "!minigames" in chat to open minigames. 1 = Yes message for aforementioned.
rps_maintenance 1 --Value 0 or 1. Toggles maintenance mode for Rock Paper Scissors minigame.
